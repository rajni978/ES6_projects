# 🚀 ES6 AI Projects

This repository contains multiple small AI-based projects built using JavaScript, Node.js, and the Gemini API.

## 📌 Features

* 💬 AI Content Generator (Explain, Summarize, Ideas, Define)
* 😂 Joke Generator
* 🌟 Affirmation Generator
* ❓ Quiz Generator

Each project uses a shared backend for secure API handling.

---

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js, Express
* API: Google Gemini API
* Version Control: Git & GitHub

---

## 🔐 Environment Setup

Create a `.env` file in the root folder:

```
GEMINI_KEY=your_api_key_here
```

---

## ▶️ How to Run Locally

### 1. Clone the repository

```
git clone https://github.com/rajni978/ES6_projects.git
cd ES6_projects
```

### 2. Install dependencies

```
npm install
```

### 3. Start the server

```
node server.js
```

### 4. Open project

Use Live Server or open `index.html` in browser.

---

## ⚠️ Important Notes

* Do not share your `.env` file
* Free Gemini API has limited requests per day
* If limit is exceeded, wait or use a new API key
